# Kishore S — Personal Portfolio

Live at: [https://portfolio-eight-weld-14.vercel.app/](https://portfolio-eight-weld-14.vercel.app/)

A cyberpunk-inspired personal portfolio website built with HTML, CSS, and vanilla JavaScript. Features glassmorphism design, interactive particle networks, and terminal HUD UI.

## Features
- 🌌 Interactive particle canvas background
- 🟩 Cyberpunk dark theme with neon lime accents
- 📱 Fully responsive design (mobile & desktop)
- 🖥️ HUD-style navigation and status bars
- 💻 Tech stack pills and interactive project cards

## Deployment
Deployed seamlessly on **Vercel**.
